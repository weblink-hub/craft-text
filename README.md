# craft-text
## 手作DIY治愈文案库
快节奏生活里，沉下心动手制作小物件是绝佳的解压方式。本文库汇总全网手工创作类随笔、配图短句、创作感悟文字，覆盖陶艺、编织奶油胶、拼豆、羊毛毡、折纸、布艺、木作、香薰、手绘石膏等各类DIY题材，适合手工爱好者发社交平台、短视频配图、写作摘抄、记录创作日常。

本仓库仅做手作文字素材资源整理，无商品带货、商业引流内容，主题独立，不和风景摄影、美食、情感短文等素材库重复，百度收录稳定，GitHub风控低。

### 一、文库手作文字分类
1. 陶艺陶瓷随笔：拉坯、彩绘陶瓷、捏陶治愈创作感悟
2. 奶油胶DIY短句：手机壳、收纳盒、小摆件制作文案
3. 羊毛毡戳戳乐：小动物挂件、立体玩偶走心记录文字
4. 布艺编织文字：毛线围巾、刺绣、帆布包手绘摘抄
5. 香薰蜡片文案：手工香薰、蜡烛、干花蜡片氛围感短句
6. 石膏娃娃手绘：涂色、改造娃娃松弛感随笔段落
7. 国风木作短文：拼图木簪、木质摆件、榫卯小物感悟
8. 亲子手工素材：儿童折纸黏土、家庭DIY温馨文字

### 二、文库使用说明
1. 全部线上站点免费浏览手作创作随笔，无付费诱导广告；
2. 区分超短句、完整创作长文，适配朋友圈、小红书、短视频；
3. 页面域名仅作为文字素材溯源查阅入口，不存在营销导流；
4. 文字温柔解压，无偏激消极内容，全年龄段均可摘抄使用。

### 三、适合阅读人群
1. DIY手工爱好者，记录制作过程、配图发社交平台；
2. 短视频图文创作者，收集手作解压类文案素材；
3. 手工店主、体验坊，撰写门店治愈宣传文字；
4. 学生、亲子家庭，积累手工主题记叙文描写段落。
